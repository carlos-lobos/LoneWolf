# LoneWolf
New Modern Program Language with AI Full Integrated

### Clonate the repository
```bash
git clone https://github.com/carlos-lobos/LoneWolf.git
```

### prerequisites
* cmake
* build-essential

### Prerequisites Installation Instructions for A Ubuntu/Debian
```bash
sudo apt install cmake
sudo apt install build-essential
```

### Build the project and run it
```bash
./build.sh
```
